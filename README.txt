== Mule Ranger Pack ==

The Mule Ranger Pack is a collection of additions to the Mule ecosystem. The following
modules are available:

 * ejetty-transport: Custom Servlet Transport, which is intended for usage when Mule is
                     running inside a Jetty container. 
 * spring-embedder : Spring FactoryBean bootstrapping of a Mule instance.
 * examples        : Example projects that makes use of the ranger pack.

Enjoy!
